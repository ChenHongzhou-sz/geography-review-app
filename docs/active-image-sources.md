## 不属于“训练配图”的内容

下面这些不是地图训练配图：

- `public/app-icon.svg`
- `public/favicon.svg`

它们是网页 APP 的图标资源，由代码手工绘制，用于浏览器标签页、主屏幕图标和 PWA 安装，不参与题目内容。

另外，页面里的大面积背景、渐变和光斑主要来自：

- CSS 渐变
- 纯样式圆形/阴影

这些也不是训练地图，不会影响题库的原图要求。

## 使用建议

如果你后续继续扩充题库，建议保持下面这条规则：

- 地图题只接教材原图或课件原图
- 非地图页面允许继续使用 CSS 背景和 SVG 应用图标
- 不把 AI 生成地图混进 `src/data/map-challenges.ts`
