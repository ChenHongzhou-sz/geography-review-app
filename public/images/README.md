# 图片目录说明

本目录当前只保留网页运行或后续单元扩展真正需要的图片资源。

## 当前重点目录

1. `七下/第七章/`
   - 亚洲单元当前正式接入主 UI 的稳定图片
   - 训练页、地图挑战页和亚洲闯关都直接引用这里的图片

2. `runtime-courseware/`
   - 课件抽取后的运行态素材库
   - 主要作为后续章节继续接入时的素材来源

3. `manifests/`
   - 已保留的图片清单与体积说明

## 当前建议

- 单元正式上线后，优先把实际使用图片整理到 `七下/第七章/` 这类稳定目录
- `runtime-courseware/` 保留为素材来源，不直接当作长期业务目录
- 不要放入 AI 生成地图、网络下载地图或手工重绘地图
